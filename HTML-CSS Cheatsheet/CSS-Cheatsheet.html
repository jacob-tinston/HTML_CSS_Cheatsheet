CSS Cheatsheet

What Is CSS?

CSS (Cascading Style Sheets) is a language that is used to style HTML and make it look appealing on the browser. It uses tags that are linked to the element in a linked HTML file with curly braces that hold the information that affects those HTML elements.

Syntax

CSS syntax is pretty simple. It takes the element/class/id from the html code, followed by curly braces ('{}') and the information inside.

CSS Properties

The properties are the parts of CSS that go inside the curly braces. These determine how the program changes the look of the HTML.
The following is a table of some common CSS properties and their meanings:

Property Meaning
color Changes color of text
text-align Moves text to left/center/right
font-family Changes the font
font-weight Changes thinkness of text
width / height Changes width / height of elements
background-color Changes color of background
margin / padding Changes space between elements
border Creates border around elements

These are just a few, but you can find the rest at https://www.w3schools.com/cssref/.

Semantic CSS

Semantic CSS basically just means that the element are named similar to what they represent. For example, if there is an <article> element that contains information about cars, the class of the article would be non-semantic if it were just named 'code'.
A semantic class would be 'cars'. Or more specifically, what part of the car the article is talking about.

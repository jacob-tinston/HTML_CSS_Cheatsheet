HTML Cheatsheet

What Is HTML?

HTML (Hypertext Markup Language) is a language used to build websites. It's essentially the backbone of everything web-based. It is a language that is primarily made up of tags that define the content shown on the webpage.

Starting Off

When starting off your HTML document, you need to tell the computer that your doing so. You do this by defining the DOCTYPE at the top of your file: <!DOCTYPE html>
This is then followed by 3 more tags:

  1. html
  2. head
  3. body

The html tag just declares that you're using the html language.
The head tag is where all of the metadata is stored. To name a few:

  • Style
  • Link
  • Script

Next, the body tag is where all of the code is; the code that is visible to the user when ran in the browser.

HTML Tags

The following is a table of some common HTML tags and their meanings:

Tag Meaning
<h1> Headings
<p> Text/paragraphs
<ul> Unordered List (Bulletpoints)
<ol> Ordered List (Numbers)
<li> Nested in <ol> / <ol> (List info)
<img> Inserts image
<strong> Bold text
<em> Italic text
<span> Separates text in <p> element
<script> Reads JavaScript inside the element

These are just a few! There are plenty more here.

Semantic HTML

Semantic HTML is just the same as html - but it's more specific and easier to read/edit.
The difference between semantic and non-semantic is:

  • non-semantic - tells nothing about content inside the element
  • semantic - defines whats inside the element
Here are some semantic html tags:

Tag Meaning
<article> Makes sections for articles
<table> Makes element for table
<header> Defines header of page
<aside> Defines unimportant info
<nav> Defines navigation bar
<footer> Defines the footer
<section> Defines other separate section

There are more elements listed here.
